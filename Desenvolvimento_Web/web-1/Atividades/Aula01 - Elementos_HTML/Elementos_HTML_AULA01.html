Manual Interativo de Elementos HTML5: Básico

Desenvolvido por Iago Correia | Baseado nos estudos de Desenvolvimento Web

Introdução e Estrutura Básica

O HTML (Hyper Text Markup Language) é a linguagem de marcação padrão utilizada para criar páginas da Web. A linguagem descreve a estrutura de uma página e consiste em uma série de elementos que informam ao navegador como exibir o conteúdo. Esses elementos são definidos por uma tag inicial, o conteúdo e uma tag final, como no exemplo: <tagname>Conteúdo</tagname>

Estrutura Básica

Salve em formato .HTML ou .html e abra no navegador.

Estrutura Básica - Elementos

  • <!DOCTYPE html> define que este documento é um documento HTML5.

  • <html> elemento raiz de uma página HTML.

  • <head> elemento que contém meta informações sobre a página HTML.

  • <title> elemento que especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página).

  • <body> elemento que define o corpo do documento e é um recipiente para todos os conteúdos visíveis, como cabeçalhos, parágrafos, imagens, etc.

O que é um Elemento HTML?

Um elemento HTML é a estrutura básica de uma página HTML. Ele é usado para marcar e organizar o conteúdo que aparece no site, como textos, imagens, links, listas, botões etc. É definido por uma tag inicial, algum conteúdo e uma tag final.

<tagname> O conteúdo vai aqui...</tagname>

O que são Atributos HTML?

São como as características ou configurações das tags. Eles fornecem as informações adicionais que o elemento precisa para funcionar corretamente.

Os 4 pontos principais:

  1. Universalidade: Todo elemento HTML pode receber atributos.

  2. Função: Eles dizem como o elemento deve agir (ex: a tag <img> precisa do atributo src para saber qual foto carregar).

  3. Onde colocar: Sempre na tag de abertura (a primeira). Nunca na de fechamento.

  4. Estrutura: Quase sempre vêm em pares de nome="valor".

O atributo lang

Você deve sempre incluir o atributo lang="pt-br" dentro da tag <html> para declarar o idioma da página da Web. Isso se destina a ajudar os motores de busca e navegadores.

Cabeçalhos (<h1> a <h6>)

O que é: São elementos usados para criar títulos ou subtítulos e organizar a hierarquia do conteúdo.

Como funciona: Os títulos são organizados em 6 níveis, indo de <h1> a <h6>. O <h1> é o mais importante, e a importância diminui sucessivamente nos níveis seguintes, até chegar ao <h6>, que é o de menor relevância.

Títulos e Parágrafos

Parágrafos (<p> ... </p>)

O que é: O elemento <p> define um parágrafo. Ele serve para estruturar blocos de texto e organizar o conteúdo do seu site.

Como funciona: O parágrafo sempre começa em uma linha nova (nunca na continuação da anterior). O navegador adiciona automaticamente uma margem (espaço em branco) antes e depois dele, para separar o texto dos outros elementos.

Linha horizontal (<hr>)

A tag <hr> define uma linha horizontal. Normalmente usada para separar o conteúdo.

A tag <hr> é uma tag vazia, o que significa que não tem tag final.

Quebras de Linha (<br>)

O elemento <br> define uma quebra de linha. Nós usamos quando queremos uma quebra de linha sem iniciar um novo parágrafo.

A tag <br> é uma tag vazia, o que significa que não tem tag final.

Formatos de Texto

O que são: São elementos usados para destacar partes do texto ou dar a ele um significado visual especial, sem precisar de estilos complexos.

Os principais

  • <b> ou <strong>: Deixam o texto em negrito (texto importante).

  • <i> ou <em>: Deixam o texto em itálico (texto enfatizado).

  • <mark>: Marca o texto com um fundo amarelo (como um marca-texto).

  • <small>: Exibe o texto um pouco menor que o normal.

  • <del>: Risca o texto (usado para indicar algo deletado ou corrigido).

  • <ins>: Sublinha o texto (usado para indicar algo inserido).

  • <sub>: Define texto subscrito (aparece abaixo da linha normal).

  • <sup>: Define texto sobrescrito (aparece acima da linha normal).

Mídias e Hiperlinks

Imagens

A tag <img> é usada para incorporar imagens em uma página web. Tecnicamente, as imagens não são inseridas no código, mas sim vinculadas a ele. A tag funciona criando um espaço reservado onde a imagem aparecerá. Ela é uma tag vazia, ou seja, contém apenas atributos e não possui tag de fechamento.

A tag <img> tem dois atributos obrigatórios:

  • O atributo src - Especifica o caminho para a imagem, pode estar online ou em um diretório do projeto.
  • O atributo alt fornece uma descrição textual para a imagem. Esse texto aparece caso a imagem não carregue e é fundamental para a acessibilidade (usado por leitores de tela para cegos).

Largura e Altura das imagens (Width e Height)

Os atributos width e height definem as dimensões (largura e altura) da imagem em pixels. No entanto, sugerimos sempre o uso do atributo style. Isso impede que folhas de estilo externas (CSS) anulem essas medidas, garantindo que a imagem permaneça exatamente com o tamanho e a proporção que você definiu.

Favicon

Um favicon é uma pequena imagem exibida ao lado do título da página na guia do navegador. Usamos o elemento <link> para inserir um favicon, após o elemento <title>.

Links (Hiperligações)

A tag <a> define um hiperlink, usado para conectar uma página a outra. O atributo mais importante é o href, que indica o destino do link (o endereço URL para onde o usuário será levado).

Links - atributo target

O atributo target especifica onde abrir o documento vinculado. Seus valores são:

  • _self: Padrão. Abre o documento na mesma janela ou guia em que foi clicado.

  • _blank: Abre o documento em uma nova janela ou guia.

  • _parent: Abre o documento no quadro pai.

  • _top: Abre o documento em todo o corpo da janela.

Listas e Itens

As listas permitem agrupar um conjunto de itens relacionados entre si. Elas servem para organizar informações, criar menus de navegação ou listar tópicos, deixando o conteúdo muito mais estruturado e fácil de ler do que um texto corrido.

Lista Não Ordenada

Uma lista não ordenada começa com a tag <ul>. Cada item da lista deve iniciar com a tag <li>. Por padrão, os itens dessa lista são marcados com pequenas bolinhas.

Lista Não Ordenada - atributo type

O atributo type da tag <ul> define o estilo do marcador de item da lista. Pode ter um dos seguintes valores:

  • disc: marcador padrão

  • circle: círculo

  • square: quadrado

  • none: sem marcador

Lista Ordenada

Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>. Ela é usada quando a ordem dos itens é importante, como em um passo a passo ou ranking.

Lista Ordenada - atributo type

O atributo type da tag <ol> define o tipo de marcador do item da lista:

  • type="1" números (padrão)

  • type="A" letras maiúsculas

  • type="a" letras minúsculas

  • type="I" números romanos maiúsculos

  • type="i" números romanos minúsculos

Lista de Descrição

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo.

  • A tag <dl> define a lista de descrição

  • A tag <dt> define o termo (nome)

  • A tag <dd> define a descrição de cada termo

Tabelas Avançadas

As tabelas permitem organizar os dados em linhas e colunas, servem exclusivamente para dados tabulares, ou seja, informações que precisam de linhas e colunas para fazerem sentido. Atenção: Use tabelas apenas para exibir informações tabulares (como listas de preços, horários ou estatísticas) e nunca para criar o design ou o layout da página.

  • <table> </table> definição da tabela
  • <th> </th> título das colunas
  • <tr> </tr> adiciona linhas na tabela
  • <td> </td> adiciona dados nas linhas

O atributo "border" define a espessura das bordas da tabela e das células

Atributo colspan

O atributo "colspan" mescla células por colunas, ou seja, para fazer uma célula abranger várias colunas.

Atributo rowspan

O atributo "rowspan" mescla células por linhas, ou seja, para fazer uma célula abranger várias linhas.

Formulários Completos

O elemento HTML <form> é um container (uma área reservada) dentro da página, responsável por agrupar campos de entrada, caixas de seleção e botões, usado para coletar dados digitados pelo usuário, geralmente é enviado a um servidor para processamento.

Eles são essenciais para a interação na web, permitindo criar telas de login, cadastros, pesquisas e páginas de contato, enviando essas informações para um servidor.

Atributos do Form

O atributo action define o destino do formulário. É nele que colocamos o endereço (URL) da página ou arquivo que receberá os dados quando o usuário clicar no botão "Enviar".

O atributo method especifica o método HTTP a ser usado ao enviar os dados do formulário.

  • GET: É o padrão. Envia os dados visíveis no endereço (URL) do navegador. Não deve ser usado para senhas.
  • POST: Envia os dados de forma oculta (no corpo da requisição). É mais seguro e ideal para dados sensíveis.

Os elementos <form>

Label e Input

O <label> define um rótulo para os elementos do form.

O <input> pode exibir vários tipos de entrada, dependendo do atributo type.

Select e Option

O elemento <select> cria uma lista suspensa (dropdown). Dentro dele, usamos a tag <option> para definir cada opção que o usuário pode escolher.

Diferença entre ID e Name

Para o formulário funcionar bem, o <select> deve ter dois atributos principais:

id: É o identificador único na página. Usado principalmente pelo CSS (para estilo) e pelo JavaScript.

name: É o identificador do dado. Quando o formulário é enviado, o servidor usa esse nome para saber qual opção foi selecionada.

Size e Multiple no elemento Select

Use o atributo size para especificar o número de valores visíveis.

Use o atributo multiple para permitir que o usuário selecione mais de um valor.

Optgroup

O elemento <optgroup> é usado para agrupar opções relacionadas dentro de uma lista suspensa (<select>).

Isso é muito útil em listas longas, pois cria categorias visuais (usando o atributo label), facilitando a leitura para o usuário.

Datalist

O elemento <datalist> fornece uma lista de opções pré-definidas para um campo <input>.

Isso cria um recurso de "autocompletar", sugerindo valores enquanto o usuário digita. Para funcionar, o atributo list do input deve ser igual ao atributo id do datalist.

Textarea

O elemento < textarea> define um campo de entrada de várias linhas

O atributo rows especifica o número visível de linhas em uma área de texto

O atributo cols especifca a largura visivel de uma área de texto

Button

o elemento <button> define um botão clicável

Fieldset e Legend

O elemento <fieldset> é usando para agrupar dados relacionados em um formulário

O elemento <legend> define uma legenda para <fieldset>

Tipos de entrada

  • <input type="button">: Botão clicável (geralmente usado com JavaScript).
  • <input type="checkbox">: Caixa de seleção (permite marcar várias opções).
  • <input type="color">: Seletor de cores (abre uma paleta de cores).
  • <input type="date">: Seletor de data (dia, mês e ano).
  • <input type="datetime-local">: Seletor de data e hora locais.
  • <input type="email">: Campo para e-mail (valida o formato automaticamente).
  • <input type="file">: Botão para selecionar arquivos para upload.
  • <input type="hidden">: Campo oculto (o usuário não vê, mas envia dados).
  • <input type="image">: Usa uma imagem como botão de enviar.
  • <input type="month">: Seletor de mês e ano.
  • <input type="week">: Seletor de semana e ano.
  • <input type="number">: Campo numérico (geralmente exibe setas para aumentar/diminuir).
  • <input type="password">: Campo de senha (esconde os caracteres digitados).
  • <input type="radio">: Botão de opção (permite escolher apenas uma opção no grupo).
  • <input type="range">: Controle deslizante (slider) para selecionar um valor.
  • <input type="reset">: Botão que limpa/reinicia o formulário.
  • <input type="search">: Campo otimizado para digitar termos de busca.
  • <input type="submit">: Botão principal que envia os dados do formulário.
  • <input type="tel">: Campo para telefone (abre teclado numérico no celular).
  • <input type="text">: Campo de texto comum de uma linha (é o padrão).
  • <input type="time">: Seletor de horário (horas e minutos).
  • <input type="url">: Campo para inserir links de sites.

Atributos de entrada

  • value: especifica um valor inicial para um campo de entrada
  • readonly: especifica que um campo de entrada é somente leitura
  • disabledespecifica que um campo de entrada deve ser desabilitada
  • sizeespecifica a largura visivel, em caracteres de um campo de entrada
  • maxiength: especifica o número máximo de caracteres permitidos em um campo de entrada
  • min e max: especificam os valores minimo e maximo para um campo de entrada
  • patternespecifica uma expressão regular com a qual o valor do campo de entrada é verificado quando o formulario é enviado
  • placeholderespecifica uma dica curta que descreve o valor esperado de um campo de entrada
  • required: especifica que um campo de entrada deve ser preenchido antes de enviar o formulario
  • autofocus especifica que um campo de entrada deve obter foco automaticamente quando a página for carregada.
